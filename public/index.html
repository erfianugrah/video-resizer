Cloudflare Video Resizer

Transform and optimize videos using Cloudflare's Media Transformation service

Quick Start

Use the following URL pattern to resize your videos:

https://cdn.erfi.dev/videos/your-video-file.mp4?width=720&quality=high

Available Options

Parameter Description Values
width Target video width Number (pixels)
height Target video height Number (pixels)
quality Video quality low, medium, high, auto
compression Compression level low, medium, high, auto
loop Enable video looping 1, 0
autoplay Autoplay video 1, 0
muted Mute audio 1, 0
